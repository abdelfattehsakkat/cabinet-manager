# front2 — Expo minimal

Démarrer le projet (dans le dossier `front2`):

```bash
cd front2
npm install
npm run web   # web
npm run ios   # iOS (mac)
npm run android # Android
```

Notes:
- Pour stocker le token, installer `@react-native-async-storage/async-storage`:
  `npm install @react-native-async-storage/async-storage`
- L'URL de l'API est l'env `REACT_NATIVE_APP_API_URL` sinon `http://localhost:3000`.
